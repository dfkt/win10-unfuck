You should remove the package for all users (-AllUsers) and also get rid of the provisioned package with this little PowerShell Script i wrote:
$appxPackageNames = (Get-AppxProvisionedPackage -Online).PackageName -like "*3d*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*alarms*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*bing*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*calc*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*camera*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*communi*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*maps*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*note*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*people*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*phone*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*photo*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*solit*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*soundrec*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*store*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*xbox*"
$appxPackageNames += (Get-AppxProvisionedPackage -Online).PackageName -like "*zune*"
foreach ($appxPackageName in $appxPackageNames) { Remove-AppxProvisionedPackage -Online -PackageName $appxPackageName }

To list all normal packages:
Get-AppxPackage -AllUsers * | ? -Property IsFramework -eq $false | sort Name | ft Name
